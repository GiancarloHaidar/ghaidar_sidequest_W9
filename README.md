## Description

This platformer encourages the player to collect all 6 carrots without losing all 3 hearts from attacking wolves and flames while they navigate the Minecraft-themed 2D sky world

---

## Learning Goals

Learning Goals:

- Replace sprite sheets
- Modify tile sets
- Link sound effects and music
